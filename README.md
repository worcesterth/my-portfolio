# Worcester — Portfolio Site

Personal portfolio and project showcase.

## Structure

```
/                   → Portfolio homepage
/split/             → หารเงินเพื่อน (Bill Splitter)
/classtrack/        → ClassTrack LINE Chatbot
/restaurant/        → Buffet Order System
```

## Deploy

Hosted on **Cloudflare Pages** — auto-deploys on every push to `main`.

## Adding a new project

1. Create a folder: `mkdir my-new-project`
2. Add `my-new-project/index.html`
3. Add a card in the root `index.html`
4. Push to GitHub → live in seconds
